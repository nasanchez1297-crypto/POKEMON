Aquí metí un catálogo oficial base y la lógica del filtro. Puedes ir ampliando el catálogo dentro del mismo archivo.
Catálogo oficial base
Edita la variable OFFICIAL_CATALOG dentro del archivo para agregar más productos oficiales y sus precios de Pokémon Center.
Cómo funciona
1. Busca el producto.
2. La app encuentra el precio oficial de Pokémon Center.
3. Calcula el máximo permitido: precio oficial + tope.
4. Sólo marca como válidas tiendas que estén dentro de ese rango.
Dónde mejorarla
El siguiente paso es conectar scraping real por tienda o una API para llenar automáticamente los precios externos. En este starter dejé algunos ejemplos simulados de tiendas para que veas el filtro funcionar.
